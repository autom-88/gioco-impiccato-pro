# 🎮 Il Gioco dell'Impiccato Pro 🎮

Hangman Pro è una versione avanzata del classico gioco dell'impiccato sviluppata con HTML, CSS e JavaScript, dotata di parole casuali, animazioni dinamiche, sistema di punteggio e interfaccia responsive.

Versione avanzata del classico gioco dell'impiccato sviluppata con HTML, CSS e JavaScript Vanilla.

## 📖 Descrizione

Il giocatore deve indovinare una parola segreta inserendo una lettera alla volta. Ad ogni errore viene mostrata una nuova parte dell'impiccato fino alla sconfitta.

Il progetto è stato sviluppato a scopo didattico per approfondire:

- JavaScript
- DOM Manipulation
- Eventi
- Array
- Variabili
- Funzioni
- Cicli
- Condizioni
- String Methods
- Math.random()
- Operatore Ternario

---

## ✨ Funzionalità

✅ Parole casuali

✅ Controllo lettere già utilizzate

✅ Contatore errori

✅ Reset automatico della partita

✅ Animazione impiccato progressiva

✅ Messaggi di vittoria e sconfitta

✅ Interfaccia responsive

✅ Codice commentato per apprendimento
